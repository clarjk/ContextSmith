# Security Policy

## Supported Versions

ContextSmith is currently in early development. Security fixes are applied to the latest released version.

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |
| Earlier versions | No |

## Reporting a Vulnerability

Please do not disclose security vulnerabilities in a public GitHub issue.

To report a vulnerability, use GitHub's private vulnerability reporting or Security Advisory feature for this repository when available. Include:

- A clear description of the issue
- Steps to reproduce it
- The affected file or feature
- The possible impact
- Any suggested fix, if known

Reports will be reviewed as soon as possible. Confirmed issues will be documented and fixed in a future release.

## Security Scope

ContextSmith processes local text and Markdown files. The current version does not call external AI APIs or send source content to third-party services. Future features involving file ingestion, web content, plugins, packaging, or external services will receive additional security review before release.
